LCORE Gen 1 (October 25, 2025)

LCORE (LetopaInteractive Creative OpenSource Runtime Engine) Gen 1 is a open source and cleaned up version of the LetopaCore (Gen 1.1) engine used in The Dark Forest: Enhanced Edition.

Engine Features:
  • Styled textHandler for colored and formatted text
  • Animated/scrolling printed text
  • Animated/scrolling inputs
    • Support to block inputs entered before input field is printed
  • Ability to find source code path, AppData path (Windows only), and bundled files in package (PyInstaller executable only)
  • Image-to-ASCII converter that can compile and play full cutscenes
    • Support for thread-accelerated compilation and rendering
